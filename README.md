# 画像分類の機械学習モデルを作成

cnn を使って料理の画像分類のモデルを生成するプログラム

ipynb ファイルは、Jupyter Notebook を使う

## リソース

foods_model/resources/ 配下に分類名(ramen, curry など)のフォルダを作成し、そのフォルダ内に画像を入れる

### image_classification_from_scratch_model

猫と犬の画像分類モデル。Keras 公式のサンプル。

### foods_model

料理の分類用モデル。
